== Description ==

一款美观的WordPress单页文件下载插件，支持自定义下载参数和样式设置。公众号：老蒋朋友圈。

## 主要功能

* 自定义内容页面中文件下载。
* 启动后下载文件附注在文章底部，美观的下载框架。
* 可自定义下载模块样式。
* 插件安装和介绍：https://www.lezaiyun.com/879.html

== Installation ==

* 1、把LeDownLoad文件夹上传到/wp-content/plugins/目录下<br />
* 2、在后台插件列表中激活LeDownLoad<br />
* 3、在左侧 设置 - LeDownLoad，菜单中设置信息。<br />
